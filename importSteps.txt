Етапи імпорту / запуску файлу у ноді:
  1. resolving - етап пошуку файлу / модулю
  Нода буде намагатися імпортувати вашу річ за таким алгоритмом
    1.1 якщо ви намагаєтеся імпортувати модуль:
      1.1.1 назва модулю шукається у вбудованих модулях ноди
      1.1.2 назва модулю шукається у node_modules
    1.2 нода вважає що ви дали шлях до файлу:
      1.2.1 шукати сам файл (якщо він з розширенням)
      1.2.2 шукає файл з розширенням .js
      1.2.3 шукає файл з розширенням .json
    1.3 нода вважає що ви дали шлях до папки:
      1.3.1 всередині папки буде шукатися файл "package.json" з властивістю "main"
      1.3.2 всередині папки буде шукатися файл index.js
      1.3.3 всередині папки буде шукатися файл index.json
    1.4 нода кидає помилку що модуль не знайдено
  2. loading - знайдений модуль нода завантажує собі в пам'ять
  3. wrapping - код з файлу з минулого кроку загортається у функцію. При цьому йому додаються 
    допоміжні змінні ноди
  4. evaluation - функцію з попереднього кроку запускають. (Для кожного конкретного файлу цей кроку
    виконується тільки 1 раз)
  5. caching - результат 4 кроку запам'ятовується та повертається при наступних імпортах
